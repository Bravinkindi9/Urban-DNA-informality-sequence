# 🧬 Urban DNA Sequencer

AI-powered urban morphology analysis using satellite building data.

## 🚀 Quick Start

### Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
```

### Deploy to Streamlit Cloud
1. Push code to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your repo
4. Deploy!

## 🎯 Features
- 🗺️ Interactive cluster visualization
- 📊 Statistical analysis
- 📥 Export to GeoJSON/CSV
- 🧬 Building morphology classification

## 🛠️ Tech Stack
- **Frontend**: Streamlit
- **Mapping**: Folium
- **ML**: scikit-learn
- **Data**: Google Earth Engine + Open Buildings V3

## 📖 Use Cases
1. **Informal Settlement Detection** - Identify high-risk areas
2. **Urban Growth Analysis** - Track city expansion
3. **Infrastructure Planning** - Target underserved neighborhoods
